## 编程与开发类

### 代码实现模板
```
请帮我实现一个[功能描述]，使用[编程语言]。要求：
1. 代码需要[具体要求，如"高效"、"简洁"、"可扩展"]
2. 实现以下功能点：[列出功能点]
3. 考虑[边界条件/异常处理/性能要求]
4. 提供详细注释
5. 给出一个使用示例

技术背景：[你的技术水平]
项目上下文：[项目背景信息]
```

### 代码优化模板
```
请帮我优化以下[编程语言]代码：

```[粘贴你的代码]```

优化目标：
1. [性能优化/可读性提升/内存占用减少/等具体目标]
2. 保持原有功能不变
3. 按照[行业标准/特定规范]重构

请解释每处优化的原因和带来的好处。
```

### 算法设计模板
```
问题描述：[详细描述算法需求]

请设计一个算法来解决上述问题，要求：
1. 时间复杂度要求：[如O(n)、O(nlogn)等]
2. 空间复杂度要求：[如O(1)、O(n)等] 
3. 提供伪代码或[首选语言]实现
4. 解释算法思路和关键步骤
5. 分析复杂度和可能的优化方向

其他约束条件：[输入规模/特殊要求等]
```
